The Automated File Organizer scans a user-selected folder, identifies files based on their extensions, and automatically moves them into appropriate folders such as Images, Documents, Videos, Music, and more. If a category folder does not exist, it is created automatically.

## рџљЂ How to Run the Project

### Prerequisites

* Python 3.x installed on your computer

### Steps

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Automated-File-Organizer.git
```

2. Navigate to the project directory:

```bash
cd Automated-File-Organizer
```

3. Run the application:

```bash
python main.py
```

4. Click **Select Folder**.
5. Choose the folder you want to organize.
6. Click **Start Organizing**.
7. The application will automatically organize your files into categorized folders.
